# MathICT AI Hub

MathICT AI Hub is an AI-powered web application for mathematics teachers and ICT coordinators. It helps with mathematics intervention planning, quiz generation, ICT support ticket tracking, school documentation, and teacher resource management.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore
- Gemini API
- NotebookLM workflow
- GitHub
- Vercel

## Main Features

- Teacher login
- Dashboard
- Mathematics performance tracker
- AI-generated intervention plans
- ICT help desk
- Runtime configuration
- NotebookLM resource references

## MVP Goals

1. Allow teachers to log in.
2. Allow teachers to create class records.
3. Allow teachers to encode assessment results.
4. Identify least mastered competencies.
5. Generate intervention plans using Gemini.
6. Allow teachers to submit ICT tickets.
7. Allow ICT coordinator to manage tickets.
8. Deploy successfully to Vercel.

## Development

```bash
npm install
npm run dev
```
